# 📝 Smart Task Manager (To-Do List Web App)

A modern and interactive **Task Manager / To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
This project was developed as **Task 1 of the Level 2 Project** during my **Web Development Internship at Oasis Infobyte**.

The application helps users manage their daily tasks efficiently with a clean and responsive user interface.

---

## 🚀 Features

- ➕ Add new tasks
- ✏ Edit existing tasks
- ❌ Delete tasks
- ✔ Mark tasks as completed
- 💾 Local Storage support (tasks remain after page refresh)
- 🎨 Modern responsive UI
- 📱 Scrollable task list

---

## 🛠 Tech Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling and UI design  
- **JavaScript (Vanilla JS)** – Application logic and DOM manipulation  
- **Local Storage API** – Data persistence
